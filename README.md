# React Приложение

Простое React приложение, созданное с использованием Vite.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

## Сборка

```bash
npm run build
```

## Предпросмотр сборки

```bash
npm run preview
```
